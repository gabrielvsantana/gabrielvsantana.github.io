Mario ─ C

A command-line software that prints # blocks that look like Mario's one.

You execute the software, and it prompts you for an integer value. It only accepts integers ranging from 1 to 8, nothing else.

Mario Less

This is the simple version with only one triangle.

expected behavior on terminal:
// Input of 8

$ ./mario
Height: 8
       #
      ##
     ###
    ####
   #####
  ######
 #######
########


// Input of 1

$ ./mario
Height: 1
#


// It only accepts from 1 to 8

$ ./mario
Height: -1
Height: 0
Height: 42
Height: 50
Height: 4
   #
  ##
 ###
####
