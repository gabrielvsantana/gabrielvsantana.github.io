Roberval ─ Scratch

It's a mini racing game I made using MIT's Scratch ─ A programming language that has a graphical user interface.

This is a simple game with only three levels. It took me about 20 hours of work to put this project together. I took all the sprites from Google Images and did some mod on Photoshop.

Instructions

  • Click on the green flag to start the game, and then, press the spacebar
  • Use the arrows to move the car
  • Complete the mission described at the beginning of each level
  • If you want to reset the game, refresh the page
  • Enjoy it! =)
