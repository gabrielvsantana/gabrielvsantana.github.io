Cash ─ C

A command-line software that checks the change value and prints out the number of coins you will need for it. It's based on a Greedy Algorithm.

The software was built to minimize the number of times the cashier would run out of coins by simply using always the least amount of it. It uses a greedy algorithm, because it achieves its purpose by always using the biggest value available.

Cash
