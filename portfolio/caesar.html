Caesar ─ C

A command-line software that encrypts the message using Caesar's algorithm.

Caesar

This is a different kind of software, as it requires a command-line argument to execute. You have to run the C program and type a key for the encryption. It can only be an integer. It uses Caesar's algorithm while retaining punctuation marks and numbers. It is also case-sensitive.

expected behavior on terminal:
$ ./caesar 13
plaintext:  HELLO
ciphertext: URYYB

$ ./caesar 1
plaintext:  HELLO
ciphertext: IFMMP

$ ./caesar 13
plaintext:  hello, world
ciphertext: uryyb, jbeyq

$ ./caesar 13
plaintext:  be sure to drink your Ovaltine
ciphertext: or fher gb qevax lbhe Binygvar

$ ./caesar HELLO
Usage: ./caesar key

$ ./caesar HELLO
Usage: ./caesar key

$ ./caesar 1 2 3
Usage: ./caesar key
