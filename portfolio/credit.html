Credit ─ C

A command-line software that checks the credit card number and it prints the credit card brand. It uses Luhn’s Algorithm.

Credit

You execute the software, and it prompts you for a long value. Using Luhn's Algorithm it checks the credit card number and outputs AMEX or MASTERCARD or VISA or INVALID.

expected behavior on terminal:
$ ./credit
Number: 4003600000000014
VISA

$ ./credit
Number: 4003-6000-0000-0014
Number: foo
Number: 4003600000000014
VISA

$ ./credit
Number: 6176292929
INVALID

The code has a few issues and the solution isn't very clean. It works for its purpose, but I need to rewrite it and it's here only for demonstration purposes.
It's been tested and perfectly works.
